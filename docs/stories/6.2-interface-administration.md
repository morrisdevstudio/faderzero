# Story 6.2 — Interface d'administration

Statut : `in-progress`

## Objectif

Afficher et gérer les membres de groupe triés par rôle (Admin > Membre > Invité), avec leurs avatars, pseudos et contrôles de départ/rétrogradation du dernier administrateur.

## Critères d'acceptation

- [x] La liste des membres est triée par ordre d'autorité (Administrateur, Membre, Invité).
- [x] L'avatar, le pseudo et le rôle sont affichés pour chaque membre.
- [x] Les administrateurs peuvent changer les rôles et retirer des membres.
- [x] Le dernier administrateur ne peut ni se rétrograder ni quitter l'espace sans nommer un autre administrateur.
- [x] Les jauges et consommations de quotas sont visibles pour les administrateurs et les membres.
