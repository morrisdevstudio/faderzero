# Story 7.4 — Interface de copie entre espaces

Statut : `in-progress`

## Objectif

Fournir une interface utilisateur ergonomique pour la copie entre espaces avec option audio décochée par défaut, calcul de taille/durée, renommage automatique `(copie N)` et verrouillage d'annulation après lancement.

## Critères d'acceptation

- [x] Dialogue de sélection de l'espace de destination.
- [x] Case à cocher audio décochée par défaut avec estimation du quota.
- [x] Nommage automatique avec le suffixe `(copie N)` en cas de conflit de nom.
- [x] Bouton d'annulation désactivé dès que la transaction est lancée.
